# Changelog

## [v1.1.0] - 2026-01-17

### UI/UX大幅改善 🎨

#### 追加機能
- **トースト通知システム**: 成功・エラー・情報通知を美しいアニメーションで表示
- **ツールチップ**: ボタンやコントロールにホバー時の説明を表示
- **カスタムアニメーション**: フェードイン、スライドイン、グロー効果など多数追加
- **輸送ルートの可視化**: アクティブな輸送ルートを青色の点線で表示
- **輸送船のアニメーション**: 移動中の物資を動くドットで表現
- **ローディング状態**: 処理中の視覚的フィードバックを追加

#### 改善
- **太陽のグロー効果**: 放射状グラデーションとパルスアニメーション
- **ホバーエフェクト**: コロニーとデポにホバー時のスケールアップとグロー効果
- **接続線のハイライト**: ホバー時に関連する接続線を強調表示
- **満足度インジケーター**: 満足度50%未満のコロニーに赤い警告ドット
- **ボタンアニメーション**: ホバー時の拡大、クリック時の縮小、シャドウ効果
- **レスポンシブデザイン**: スマートフォンとタブレットに最適化
- **ローディング画面**: 太陽系初期化中の美しいローディング表示

#### 技術的改善
- Tailwind CSS カスタムアニメーション追加
- TypeScript 型安全性の向上
- コンポーネント分離（Toast, Tooltip）
- パフォーマンス最適化
